# Repomix 運用ガイド

## Repomix とは？

[Repomix](https://repomix.com/)は、リポジトリのコードベースを、AI（LLM）が解析しやすいように単一のファイルにまとめるためのCLIツールです。

## なぜ導入するのか？

AIアシスタント（例: Gemini, ChatGPT, Claude）にプロジェクトの全体像を正確に把握させ、以下のようなタスクを効率的に依頼することを目的としています。

- 機能追加や変更に関する相談
- コードレビュー
- リファクタリング
- ドキュメント生成
- バグ修正

`repomix`が生成したファイルを提供することで、AIはコードベースの文脈を深く理解し、より精度の高い応答を返すことが期待できます。

## 使い方

### 1. インストール

`repomix`を使用するには、[Node.js](https://nodejs.org/)が必要です。<br>
以下のコマンドを実行することで、`repomix`をグローバルインストールできます。

```shell
npm install -g repomix
```

### 2. 設定ファイル

`repomix`の設定は、プロジェクトルートの`repomix.config.json`で一元管理しています。<br>
出力形式や対象ファイルなどを定義しており、通常、このファイルを変更する必要はありません。

### 3. 実行

プロジェクトのルートディレクトリで、以下のコマンドを実行するだけです。

```shell
repomix
```

### 4. 出力

コマンドを実行すると、プロジェクトルートに`repomix-output.md`というファイルが生成されます。<br>
なお、このファイルは`.gitignore`で、コミット対象から除外しています。

AIアシスタントに何かを依頼する際は、この`repomix-output.md`の内容をコピーし、プロンプトに貼り付けて使用してください。<br>
特に、無料で利用しやすいWeb版のGeminiやGoogle AI Studioでは、この単一ファイルをアップロードするだけでプロジェクト全体のコンテキストを簡単に提供でき、非常に便利です。
